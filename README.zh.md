# logseq-plugin-favorite-tree

## 功能

- 基于 namespace 或者页面属性实现树形“收藏”与“最近使用”。可通过`fixed`页面属性来调整展示顺序，例如 `fixed:: 100`，数值越小位置越靠前。
- 可在页面上通过属性设置组合过滤器，会在树形收藏上展示。可参见下方演示视频。
- 通过拖拽来调整左侧边栏宽度。
